# Kanity
Simple RPG game engine.

## How to build
1. `$ git clone git@github.com:Chikuwabu/Kanity.git` or `$ https://github.com/Chikuwabu/Kanity.git`
2. `$ cd Kanity`
3. `$ dub build`

## How to run
`$ ./bin/kanity`
or
`$ dub run`
## Build dependencies
- git
- dub
- SDL 2.0 (sdl2)
- SDL_image 2.0 (sdl2_image)

## Run dependencies
- SDL 2.0 (sdl2)
- SDL_image 2.0 (sdl2_image)
